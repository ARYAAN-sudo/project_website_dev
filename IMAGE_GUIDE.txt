## Step 2: Prepare Your Images

### Recommended Image Sizes:
- **Hero/Header images**: 1920x1080 pixels (landscape)
- **Salt type images**: 600x400 pixels (landscape)
- **Production process images**: 800x600 pixels
- **Region images**: 800x500 pixels
- **Scheme images**: 600x400 pixels
- **Map image**: 1000x600 pixels

## Step 3: Add Images to the Folder

1. Copy all your images into the `images` folder
2. Make sure image names match exactly (case-sensitive)
3. Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`

## Step 4: Update HTML Files (if using different names)

If your image names are different, update the HTML files:

### Example in salt-types.html:
Find this line:
```html
<img src="images/sea-salt.jpg" alt="Sea Salt">
```

Change `sea-salt.jpg` to your image name:
```html
<img src="images/YOUR_IMAGE_NAME.jpg" alt="Sea Salt">
```

## Step 5: Optimize Your Images (Optional but Recommended)

For faster loading:
1. **Compress images**: Use online tools like:
   - TinyPNG (tinypng.com)
   - Compressor.io
   - ImageOptim

2. **Convert to WebP**: WebP images are smaller:
   - Use Squoosh.app
   - Or online converters

3. **Resize large images**: Don't use 4K images for web
   - Maximum 1920px width for hero images
   - 800px width for content images

## Step 6: Test Your Website

1. Open `index.html` in your browser
2. Click through all pages
3. Check if all images load correctly
4. If image doesn't show:
   - Check the file name spelling
   - Check the file is in `images` folder
   - Check the file extension (.jpg, .png, etc.)

## Fallback Images

Don't worry! The website has **automatic fallback** to Unsplash images if your images are not found. The website uses this code:

```html
<img src="images/your-image.jpg" onerror="this.src='https://images.unsplash.com/...'">
```

This means:
- If `images/your-image.jpg` exists ‚Üí Shows your image ‚úÖ
- If it doesn't exist ‚Üí Shows Unsplash image ‚úÖ
- Website never shows broken images! üéâ

## Tips for Best Results

1. **Use high-quality images** - Clear, well-lit photos
2. **Consistent style** - All images should have similar look
3. **Relevant content** - Images should match the section
4. **Proper aspect ratio** - Follow recommended sizes
5. **Compress before upload** - Faster website loading

## Need Help?

- Images not showing? Check browser console (F12) for errors
